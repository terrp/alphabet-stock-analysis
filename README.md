# Alphabet Inc. (GOOG) Stock Analysis

## Overview
Time-series analysis of Alphabet stock prices (Apr 1 - Oct 1, 2020) identifying trends and moving average crossovers.

## Questions Addressed
- Q1(i): Identify two significant price trends
- Q1(ii): Analyze 7-day and 30-day moving average crossovers

## Quick Start

### Requirements
- Python 3.x
- pandas, numpy, matplotlib
- Jupyter notebook

### Installation
```bash
python3 -m venv venv
source venv/bin/activate
pip install pandas numpy matplotlib jupyter
```

### Run Analysis
```bash
jupyter notebook analysis.ipynb
```
